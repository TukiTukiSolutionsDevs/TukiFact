## Qué Incluye Cada Plan

### Free (S/0)
- 50 documentos/mes
- Factura + Boleta + Notas
- Panel web básico
- Sin API
- Sin soporte
- Sin IA
- 1 usuario

### Emprendedor (S/39)
- 300 documentos/mes
- Factura + Boleta + Notas + Resumen + Baja
- Panel web completo
- API REST (100 req/min)
- Soporte email
- Sin IA
- 3 usuarios
- 1 serie por tipo

### Negocio (S/79)
- 1,000 documentos/mes
- Todo lo anterior
- API REST (300 req/min)
- Webhooks
- Soporte email + tickets
- IA básica (consultas simples)
- 10 usuarios
- Múltiples series
- Logo y colores personalizados

### Profesional (S/149)
- 3,000 documentos/mes
- Todo lo anterior
- API REST (500 req/min)
- IA completa (Agente Emisor)
- BYOK (traer tu propia API key de IA)
- Soporte prioritario
- 25 usuarios
- Reportes avanzados

### Empresa (S/299)
- 10,000 documentos/mes
- Todo lo anterior
- API REST (1000 req/min)
- IA completa con todos los agentes
- Múltiples puntos de emisión
- Usuarios ilimitados
- API dedicada
- SLA 99.9%

### Developer (S/99 + S/0.05/doc extra)
- 1,000 documentos incluidos
- API REST completa (500 req/min)
- Sandbox permanente
- SDKs (Python, JS, C#)
- Documentación premium
- Webhooks
- IA como copilot de integración
- Sin panel web (API-only)
- 5 usuarios técnicos

## Modelo de Monetización Adicional
- **Documentos extra**: S/0.05 - S/0.10 por doc según plan
- **Almacenamiento extra**: S/5/mes por cada 1GB adicional
- **API Keys adicionales**: S/10/mes cada una
- **Certificado digital**: Pass-through del proveedor + margen

## Estrategia de Precio
1. **Free** captura mercado y genera boca-a-boca
2. **Emprendedor** compite directo con eFact y EmitePeru
3. **Negocio** compite con NubeFact ofreciendo IA que ellos no tienen
4. **Developer** es un OCEAN BLUE — nadie en Perú tiene un plan así
5. **Profesional/Empresa** monetiza a empresas que necesitan más
